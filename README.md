# 🔍 AI Scan

An interactive web application that uses OpenAI's GPT-4 Vision API to analyze and describe images in real-time.

## Features

✨ **Image Upload** - Drag and drop or click to upload images
📷 **Webcam Capture** - Use your device's camera to capture photos
🤖 **AI Analysis** - Get detailed descriptions of what's in your images
🔒 **Secure** - API key stored locally in your browser
📱 **Responsive** - Works great on mobile and desktop devices
⚡ **Fast** - Real-time analysis with OpenAI's GPT-4 Vision

## Getting Started

### Prerequisites

- An OpenAI API key with GPT-4 Vision access
